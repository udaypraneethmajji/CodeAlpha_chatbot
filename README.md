# CodeAlpha AI Internship - Chatbot

## 📌 Project Description
This is a simple FAQ chatbot built using Python and NLP techniques.

## 🛠 Technologies Used
- Python
- Scikit-learn

## 🚀 Features
- Answers user questions
- Uses cosine similarity to find best match

## ▶️ How to Run
1. Install required libraries:
   pip install scikit-learn

2. Run the program:
   python chatbot.py
